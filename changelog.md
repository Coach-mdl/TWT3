CHANGES:

A complete rewrite of the pack, reducing bloat and adding things that probably should have been around from the start.
There will be a server wipe with this update. The changes are too large to simply leave the world as is.

- TNT now uses TFC sand
- Create chocolate now has food data.
- Added a plethora of structures to explore
- The end is now accessible. Have fun breaking the game (Not that there's much there).
- Added a new /tfcdate command which shows you the current date and time left until next season.
- Fixed sequenced recipes for various glassworking products.
- Nether brick can no longer collapse.
- Metal blocks are now crafted with four sheets and a smooth stone block. (Provided it has a sheet)
- Most recipes that use planks blocks have been switched to use lumber where applicable.
